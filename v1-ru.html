Обо мне

Frontend-разработчица с уклоном в верстку

Большое внимание в своей работе я уделяю созданию дружелюбных, понятных и отзывчивых пользовательских интерфейсов. Тесно сотрудничаю с дизайн-отделом и верстаю полностью адаптивные страницы даже в случаях, когда в макете не представлен дизайн для мобильных устройств. Придерживаюсь подходов BAM, Atomic Design и Storybook Driven Development, это позволяет мне создавать дизайн-системы, которые легко поддерживать и масштабировать.

Обращаю внимание на детали и умею работать согласно техническим заданиям, имею опыт работы в команде с англоязычными коллегами. Знаю Git, SASS, Tailwind и CSS-modules, React, имею опыт создания анимаций, изучаю React Native и TypeScript.

Опыт работы

Pixel Point Frontend-разработчица

Участвую в разработке сайтов для ряда технологических компаний с использованием Gatsby, Next.js и Tailwind.

Важную часть моей работы составляет разработка интерфейсов, создание качественной адаптивной кроссбраузерной верстки, добавление новой функциональности и интеграция страниц с CMS. Принимала участие в корпоративных хакатонах.

Isovalent — занимается разработкой open-source ПО и корпоративных решений, флагманская технология компании — Cilium — используется Google, GitLab, Adobe.

Занимаюсь разработкой библиотеки ресурсов, системы поиска и фильтров для нее. Разрабатываю новые шаблоны страниц, улучшаю существующие, занимаюсь технической поддержкой клиента. Также работаю над сайтами дочерних продуктов компании:

  • eBPF — технология, которая позволяет запускать произвольный код пользователя в рамках ядра.
  • Cilium — open source проект, который обеспечивает сетевые, безопасностные и мониторинговые функции для облачных сред, включая Kubernetes и другие платформы оркестрации контейнеров.
  • Tetragon docs – документация для компонента Cilium на базе eBPF, который позволяет в реальном времени наблюдать за безопасностью и контролировать выполнение программного обеспечения.

Configu — платформа управления конфигурацией E2E приложений. Я участвовала в подготовке проекта к релизу, имплементировала анимации, разрабатывала раздел документации.

Fonoa — платформа автоматизации налогообложения, упрощающая управление налогами по всему миру. Я участвовала в подготовке проекта к релизу, разрабатывала раздел Products.

Также работала над проектами RevenueCat и Neon.
